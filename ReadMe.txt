Программа решает следующую задачу:

Преобразование исходного бинарного файла в текстовые файлы формата *.mif.
При преобразовании должны создаваться N *.mif файлов с именами типа abc1.mif...abcN.mif где abc это имя исходного файла.
Файлы должны заполняться поочередно. Недостающие слова заполняются как FF.

Настраиваемые параметры:
1) Количество итоговых файлов
2) Разрядность данных в битах
3) Количество слов в каждом файле

Тестовый файл:
test_file.bin
